# Pacemaker Website

Figma 디자인 기반 북미 취업 준비 플랫폼 랜딩 페이지입니다.

## 구조

- `index.html` - 메인 정적 HTML 페이지
- `images/` - 이미지 assets
- `design-system.md` - 디자인 시스템 문서
- `services-v3-en.html` - 서비스 페이지 참고용
- `export/` - HTML + 이미지 정리 폴더

## 실행 방법

`index.html`을 브라우저에서 열거나, 로컬 서버로 실행하세요:

```bash
npx serve .
```

또는 `python -m http.server` 등 정적 서버를 사용할 수 있습니다.
